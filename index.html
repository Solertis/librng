libRNG by Brendan Hickey

A public domain collection of efficient and robust random number generators.

Generators

  • KISS
    • KISS99
    • JKISS
    • JKISS32
    • JLKISS64
    • SuperKISS
  • Xorshift
    • 32-bit <13,17,5>
    • 64-bit <13,7,17>
    • 64-bit templated
  • A buffered RNG

Dependencies

libRNG has been tested with:

  • gcc v4.2.1, v4.3.2
  • GNU make v3.81
  • Debian 5.0, Ubuntu 10.10, OS X 10.6.6

Install

Build a shared object with `make`
